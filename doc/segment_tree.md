## Description:
Segment tree with fast optimization and based on
Atcoder segment tree

## Usage:

* define op() for operation and e() for identity element
* SegTree<T>st(size,op,e)

## Time:
* $O(n)$ for build()
* $O(log(n))$ for add(),modify(),query()
